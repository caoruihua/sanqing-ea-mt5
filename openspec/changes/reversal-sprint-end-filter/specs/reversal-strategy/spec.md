## MODIFIED Requirements

### Requirement: 反转信号入场条件

系统必须在满足以下所有条件时才生成反转信号：
1. 明显涨跌前置条件：2小时内涨跌幅 >= ATR × 2.8
2. 趋势方向条件：处于上涨趋势（看跌反转）或下跌趋势（看涨反转）
3. K线形态条件：乌云盖顶、长上影线（看跌反转）或长下影线（看涨反转）

#### Scenario: 看跌反转信号生成
- **WHEN** 2小时涨幅 >= ATR × 2.8
- **AND** 处于上涨趋势
- **AND** 出现乌云盖顶或长上影线形态
- **THEN** 系统生成看跌反转信号

#### Scenario: 看涨反转信号生成
- **WHEN** 2小时跌幅 >= ATR × 2.8
- **AND** 处于下跌趋势
- **AND** 出现长下影线形态
- **THEN** 系统生成看涨反转信号

#### Scenario: 无明显涨跌时不入场
- **WHEN** 2小时涨跌幅 < ATR × 2.8
- **THEN** 系统不生成任何反转信号

### Requirement: 止盈止损计算

系统必须按以下方式计算止盈止损：
- 看跌信号：止损 = high24 + 6美元，止盈 = 入场价 - ATR × 2.5
- 看涨信号：止损 = low24 - 6美元，止盈 = 入场价 + ATR × 2.5

#### Scenario: 看跌信号止盈止损
- **WHEN** 生成看跌反转信号
- **THEN** 止损 = high24 + 6.0
- **AND** 止盈 = entry - ATR × 2.5

#### Scenario: 看涨信号止盈止损
- **WHEN** 生成看涨反转信号
- **THEN** 止损 = low24 - 6.0
- **AND** 止盈 = entry + ATR × 2.5

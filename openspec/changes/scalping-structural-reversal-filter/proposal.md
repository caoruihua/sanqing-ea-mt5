## Why

当前剥头皮 EA 在日内行情已经明显延伸后仍可能继续追单，尤其是 `StrongTrendRatio` 强趋势直接入场或连续 EMA 同侧入场逻辑在局部高点/低点附近触发时，容易买在天花板或卖在地板。

本变更新增“结构性反转危险区过滤器”，在疑似日内转折区域跳过同方向剥头皮开仓，优先减少被套和追在末端的风险。

## What Changes

- 在 `mt5/剥头皮脚本.mq5` 中新增下单前的结构性反转危险区检查。
- 当行情近期快速上涨、价格接近近期高位区间、并出现看跌反转证据时，禁止继续开多单。
- 当行情近期快速下跌、价格接近近期低位区间、并出现看涨反转证据时，禁止继续开空单。
- 当价格有效突破该结构性危险区后，解除对应方向的禁入限制，允许原有顺势逻辑继续开单。
- 保持当前策略定位：只过滤危险的顺势追单，不新增反手交易。
- 新增可配置参数，包括冲刺行情回看窗口、ATR 倍数阈值、高低位区间比例、反转 K 线检查和调试日志。
- 保留现有日内风控、时间控制、持仓检查、止盈和止损逻辑。

## Capabilities

### New Capabilities

- `scalping-structural-reversal-filter`：识别日内结构性反转危险区，并拒绝该区域附近的同方向剥头皮追单。

### Modified Capabilities

- 无。

## Impact

- 影响代码：
  - `mt5/剥头皮脚本.mq5`
- 影响行为：
  - 当反转危险区过滤器触发时，当前 EMA、回调、强趋势或持续趋势逻辑产生的部分买入/卖出信号会被跳过。
  - 当价格突破危险区并确认趋势延续时，过滤器不再阻止该方向的顺势开仓。
- API / 依赖：
  - 不新增外部依赖。
  - 不改变券商接口或交易 API 契约。
- 运行影响：
  - 在日内行情延伸末端，交易频率预计会降低。
  - 启用调试日志时，应输出候选交易被跳过的原因。
